acpme {regimes}

Adjustment for Confounding in the Presence of Multivariate Exposures

See tech report for here: Model-Averaged Confounder Adjustment for Estimating Multivariate Exposure Effects. Part of the regimes R package

An Example

Load the regimes package, simulate data, and fit ACPME.

#load package
library(regimes)

#simulate data
dat <- simregimes(scenario="acpme1", seed=1234)

#estimate model
fit <- acpme(Z=dat$Z,C=dat$C,y=dat$Y, niter=1000)

Summarize results

#summerize
summer <- summary(fit)
